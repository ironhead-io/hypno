Summary

libhypno is a way to design websites, mobile APIs and other internet applications with Lua and C. It aims to be simple to setup on any platform, and low on memory.

Setup

libhypno requires C, Lua 5.3 or greater, a C compiler and any recent version of make. The most recent version can be downloaded from this site here or via your terminal.

$ wget -O libhypno.tgz http://libhypno.com/latest.tgz

Optional: libhypno is also maintained via Git and can be pulled from its home on Github.

$ git clone https://github.com/cdesign/hypno

Once the package is present on your system, you can decompress and run `make` to get a build going.

tar xzf libhypno.tgz && cd libhypno && make

Usage

libhypno is a way to design websites, mobile APIs and other internet applications through the use of a simple webserver and Lua. It aims to be simple to setup on any platform, and low on memory.
